# 🧠 Abstracts

This folder contains **non-CSS generating SASS files**.

## Includes

- Variables
- Mixins
- Functions

## Purpose

These files are used as **helpers** for styling but do not generate CSS directly.

## Example

- \_variables.scss → color, spacing, typography tokens
- \_mixins.scss → reusable logic like flex-center

## Rule

❗ Do NOT write actual styles here
